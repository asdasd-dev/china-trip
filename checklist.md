## Что взять с собой

### Чемодан (сдаваемый багаж)
- [ ] Одежда на 15 дней (или стирка в отеле): футболки 5–6 шт, шорты/брюки 3–4 шт
- [ ] Лёгкая куртка/ветровка (Чжанцзяцзе, Великая стена — ветер на высоте)
- [ ] Треккинговые кроссовки (Чжанцзяцзе, стена — обязательно удобная обувь)
- [ ] Кепка/панама + солнцезащитный крем SPF 50
- [ ] Дождевик компактный (май — сезон дождей в Чжанцзяцзе)
- [ ] Зарядки: телефоны, наушники, powerbank кабели
- [ ] Адаптер розетки не нужен — в Китае розетки подходят для евровилки
- [ ] Купальник (если бассейн в отеле)
- [ ] Пижама / домашняя одежда

### Для девушки дополнительно
- [ ] Удобная обувь на каждый день (не каблуки — много ходьбы!)
- [ ] Платье / нарядная одежда для Flair Rooftop и Da Dong (smart casual)
- [ ] Солнцезащитные очки
- [ ] Косметика, уходовая косметика (в дорожных объёмах до 100 мл для ручной клади)

### Ручная кладь (рюкзак)
- [ ] Паспорта (2 шт)
- [ ] Распечатки броней отелей (на всякий случай)
- [ ] **Powerbank** (обязательно в ручной, не в багаже!)
- [ ] Наушники
- [ ] Телефоны + зарядки
- [ ] Перекус на перелёт
- [ ] Кофта/худи (в самолёте холодно)
- [ ] Маска для сна + беруши (ночной перелёт)
- [ ] Влажные салфетки
- [ ] Наличные юани ¥1 500–2 000

### На каждый день (дневной рюкзак)
- [ ] Телефон + powerbank + кабель
- [ ] Бутылка воды (пополнять из кулеров в отеле)
- [ ] Кепка + SPF
- [ ] Дождевик (компактный)
- [ ] Наличные ¥200–500 (резерв)
- [ ] Паспорта (нужны для входа в достопримечательности!)
